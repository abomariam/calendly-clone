# Task 05: Build Event Summary Panel

## Goal

Create the left-side event summary panel shown throughout the booking flow.

## Steps

- Add a static avatar asset or static image reference.
- Build `EventSummary` with:
  - avatar
  - event name
  - duration
  - description
  - selected date and time when a slot has been chosen
  - selected timezone when relevant
- Add a back button only on the details form step, returning to date/time selection.
- Omit unsupported screenshot links and branding.

## Acceptance Criteria

- The summary panel matches the screenshots closely on desktop.
- The selected slot appears in the summary on the details form step.
- The summary stacks cleanly above the workflow area on mobile.
